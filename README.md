# 競技プログラミング pythonライブラリ
## 構成
- math 数学系
  - gcd 最大公約数
  - lcm 最小公倍数
  - factorial 階乗
  - mod mod処理
  - binomial_coefficient 二項定理
  - divisor 約数
  - prime_factorization 素因数分解
  - matrix_product 行列積

- dp　動的計画法の一般化
  - bitDP bitDP
  - digitsDP 桁DP
  - sectionDP 区間DP
- graph グラフ
  - unionfind Union-Find

  - wardhallfloyd ワーシャルフロイド法
  - prim プリム法

- search 全探索

  - dfs 深さ優先探索
  - binary_search 二分探索
  - ternary_search 三分探索
  - bit_search bit全探索

- section 区間処理
  - imos いもす法

  - coordinate_compression 座標圧縮
  - segmenttree セグメント木

- other その他

- heuristic ヒューリスティックコンテスト
